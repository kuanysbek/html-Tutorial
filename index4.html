Тег <img ⁄> 24.07.2022

Тег используется для вставки изображения. Он содержит только атрибуты и не требует закрывающего тега.

Ссылка на изображение (веб-адрес) добавляется с помощью атрибута src.

Синтаксис для добавления изображения выглядит следующим образом:

<img src="image.jpg" ⁄>

Атрибут alt устанавливает альтернативный текст для изображений.

Блакнот
